🎵 Spotify Clone – Project Description

The Spotify Clone is a front-end web application built using HTML, CSS, and JavaScript that replicates the basic design and functionality of the popular music streaming platform Spotify. This project focuses on creating an interactive and visually appealing user interface along with essential audio playback features.

🚀 Features
🎧 Music Player Controls
Play, pause, next, and previous song functionality.
📃 Song List Display
A list of songs with titles, artists, and duration.
⏱️ Progress Bar
Shows current playback time and allows users to seek through the track.
🔊 Volume Control
Adjust audio levels dynamically.
🖼️ Modern UI Design
Inspired by Spotify’s layout with sidebar navigation and album sections.
❤️ Interactive Elements
Buttons, hover effects, and animations using CSS and JavaScript.
🛠️ Technologies Used
HTML – Structure of the web pages
CSS – Styling, layout, and responsiveness
JavaScript – Functionality and interactivity (audio controls, DOM manipulation)
