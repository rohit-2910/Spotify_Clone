🎵 Spotify Clone

A simple Spotify-inspired music player web app built using HTML, CSS, and JavaScript. This project replicates the basic UI and core functionalities of Spotify, allowing users to play and control music directly in the browser.

🚀 Features
▶️ Play / Pause music
⏭️ Next / Previous track
📃 Song list with details
⏱️ Interactive progress bar (seek functionality)
🔊 Volume control
🎨 Modern and responsive UI
✨ Smooth animations and hover effects
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling and layout
JavaScript (ES6) – Functionality and interactivity
